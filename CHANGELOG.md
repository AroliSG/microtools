# Changelog

All notable changes to this project are documented in this file.

## v1.0.0 - 2026-02-16

### Added
- Multi-tool Discord suite with categorized navigation.
- Asset CDN helper with hash resolution and image download flow.
- ID Lookup with snowflake decoding and JSON output.
- Emoji Maker with Discord-size output checks.
- Embed Creator with import/export JSON.
- Markdown Helper, Invite Generator, Permission Calculator, Timestamp and Snowflake tools.
- Webhook sender tool.
- Microtools landing with search and collection routing.
- Light/dark mode toggle.
- SEO essentials: `robots.txt`, `sitemap.xml`, canonical tags, social preview asset.

### Changed
- Rebranded project naming to **Microtools**.
- Refactored major tools to shared UI primitives for consistency and maintainability.

### Notes
- Deployment currently done manually by uploading `dist/` to VPS.
